## Instalación

El primer paso es clonar el proyecto usando git.

> Si no tenés experiencia previa usando git, te va a convenir usar [la interfaz gráfica de github](https://desktop.github.com/) y [hacer los tutoriales introductorios](https://lab.github.com/).

Para clonarlo usando la terminal podés hacerlo con:

``` bash
git clone https://github.com/gcba/Obelisco.git
```

Para instalar las dependencias, ingresá a la carpeta del proyecto y corré:

``` bash
npm install
```

## Puesta en marcha

Una vez instalado, se puede iniciar el proyecto con el siguiente comando.

```bash
npm start
```

Después de algunos (varios) segundos se te va a abrir la página del proyecto corriendo de forma local en tu computadora.
Si esto no pasó, podés ingresar en [http://localhost:6006/](http://localhost:6006/). -->
