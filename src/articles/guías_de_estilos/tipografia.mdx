## Usos

<Row>
<Col items={2}>

### Títulos

Se utilizan para crear niveles de lectura jerárquicos.

</Col>
<Col items={2}>
<Example align="center">

<h1>Título 1</h1>
<h2>Título 2</h2>
<h3>Título 3</h3>
<h4>Título 4</h4>
<h5>Título 5</h5>
<h6>Título 6</h6>

</Example>
</Col>
</Row>

<Row>
<Col items={2}>

### Subtítulos

Se utilizan para acompañar a los títulos, agregando información con un nivel de jerarquía menor.

</Col>
<Col items={2}>
<Example align="center">
  
<div className="lead">Estilo de subtítulo</div>

</Example>
</Col>
</Row>

<Row>
<Col items={2}>

### Cuerpos de texto

Se utilizan para todos los cuerpos de lectura extensa y como principal variable texto.

</Col>
<Col items={2}>
<Example align="center">

Open Sans Regular 16px

<small>Open Sans Regular 13px</small>

</Example>
</Col>
</Row>

<Row>
<Col items={2}>

### Volantas, etiquetas, fechas

Se utilizan para información adicional o elementos pequeños.

</Col>
<Col items={2}>
<Example align="center">

<span class="small">SOBRELÍNEA</span>

</Example>
</Col>
</Row>

<Row>
<Col items={2}>

### Epígrafes

Se utiliza para información adicional en fotos, o detalles en el texto.

</Col>
<Col items={2}>
<Example align="center">

<p className="text-xs">Estilo para epígrafe</p>

</Example>
</Col>
</Row>

<Row>
<Col items={2}>

### Destacados y citas

Se utiliza para citas, mencionando a su autor, o para enfatizar una frase o una idea dentro de un texto.

</Col>
<Col items={2}>
<Example align="center">

<blockquote>
    <p>
      “Cada acción cuenta, si hacemos las cosas bien podemos salvar vidas y cuidarnos entre todos”
    </p>
    <small>Horacio Rodríguez Larreta</small>
  </blockquote>

</Example>
</Col>
</Row>

## Formatos de texto

<Row>
<Col items={2}>

### Bold o Negrita

Se utilizan para dar jerarquía dentro de una oración. Se admite este recurso solo en subtítulos, cuerpos de textos, volantas y etiquetas.

</Col>
<Col items={2}>
<Example>

La Ciudad creó una **nueva normativa** de estacionamiento para que los vecinos entiendan fácilmente dónde...

</Example>
</Col>
</Row>

<Row>
<Col items={2}>

### Itálica

Se utiliza para destacar un texto con menor jerarquía que negrita. 
Se admite este recurso solo en subtítulos y cuerpos de textos.

</Col>
<Col items={2}>
<Example>

La Ciudad creó una _nueva normativa_ de estacionamiento para que los vecinos entiendan fácilmente dónde...

</Example>
</Col>
</Row>

## Diseño y estructura (contexto de uso)
La extensión es muy importante a la hora de interpretar un texto, mantenga los párrafos concisos. Algunos usuarios con discapacidades cognitivas (e incluso aquellos que no las tienen) pueden tener muchas dificultades para leer y comprender grandes bloques de texto.

- Proporcione un título descriptivo y claro.
- Use encabezados y subencabezados para jerarquizar el contenido para que las secciones sean claramente identificables.
- Use los tamaños de fuentes definidas ya que aseguran legibilidad.

## Usabilidad

### Buenas prácticas

[[si | Respetar los estilos, su orden, espaciado, tamaño y paleta cromática. ]]

[[si | Utilizar mayúscula solo al iniciar una oración. ]]

[[si | Utilizar contenido breve y sencillo. ]]

### Malas prácticas

[[no | No modificar los estilos tipográficos como el tamaño, la lineatura o la interlínea. ]]

[[no | No modificar el orden de los estilos. ]]

[[no | No usar subrayados en encabezados. ]]

[[no | No alinear el contenido al centro o a la derecha. No utilice sangrías. ]]
