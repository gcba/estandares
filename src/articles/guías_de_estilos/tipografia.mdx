## Usos

<Row><Col items={2}>

### Encabezados
Los encabezados se utilizan para crear niveles de lectura jerárquicos.

</Col><Col items={2}><Example align="center">

<h1>Título 1</h1>
<h2>Título 2</h2>
<h3>Título 3</h3>
<h4>Título 4</h4>

</Example></Col></Row>

<Row><Col items={2}>

### Subtitulos
Se utilizan para acompañar a los encabezados, agregando información con un nivel de jerarquía menor.

</Col><Col items={2}><Example align="center">
<div className="lead">Subtítulo 1</div>
</Example></Col></Row>

<Row><Col items={2}>

### Cuerpos de texto
Se utiliza para todos los cuerpo de lectura extensa, y como principal variable texto.

</Col><Col items={2}><Example align="center">

Cuerdo de texto
<small>Cuerdo de texto</small>

</Example></Col></Row>

<Row><Col items={2}>

### Llamados a la acción
Se utiliza únicamente para los botones.

</Col><Col items={2}><Example align="center">

<button className="btn btn-link">Botón</button>

</Example></Col></Row>

<Row><Col items={2}>

### Detalles
Se utiliza para información adicional o elementos pequeños.

</Col><Col items={2}><Example align="center">

<small>OVERLINE</small><br />
<small>Caption</small>

</Example></Col></Row>

## Formatos de texto

<Row><Col items={2}>

### Bold o Negrita
Se utilizan para dar jerarquía dentro de una oración. Se admite este recurso solo en subtítulo, body y caption.

</Col><Col items={2}><Example>

Lorem ipsum dolor sit amet, consetetur **sadipscing elitr**, sed diam nonumy eirmod tempor invidunt ut.

</Example></Col></Row>

<Row><Col items={2}>

### Italica o Cursiva
Se utiliza para destacar un texto o una cita, con menor jerarquía que negrita. Se admite este recurso solo en subtítulo, body y caption.

</Col><Col items={2}><Example>

Lorem ipsum dolor sit amet, consetetur *sadipscing elitr*, sed diam nonumy eirmod tempor invidunt ut.

</Example></Col></Row>

<Row><Col items={2}>

### Destacado
Se utiliza para enfatizar una frase o una idea dentro de un texto. Se admite este elemento solo dentro de un body 1 o body 2. Su estilo de carácter es Subtitulo 2.

</Col><Col items={2}><Example>

> Lorem ipsum dolor sit amet, consetetur sadipscing elitr.

</Example></Col></Row>

## Practicas de uso

<Row><Col items={2}>

### Estilos tipográficos.
Respetar los estilos, **su orden, espaciado, tamaño y paleta cromática**.

</Col><Col items={2}>

![completar](/guía-de-estilos/tipografía/bueno-uso-1.svg)

</Col></Row>

<Row><Col items={2}>

### Mayúsculas.
Utilizar **mayúscula** al iniciar una oración.

</Col><Col items={2}>

![completar](/guía-de-estilos/tipografía/bueno-uso-2.svg)

</Col></Row>

<Row><Col items={2}>

### Contenido.
Utilizar contenido breve y sencillo.

</Col><Col items={2}>

![completar](/guía-de-estilos/tipografía/bueno-uso-3.svg)

</Col></Row>

<Row><Col items={2}>

### Estilos tipográficos.
No modificar los estilos tipográficos, como **el tamaño, la lineatura o la interlinea**.

</Col><Col items={2}>

![completar](/guía-de-estilos/tipografía/mal-uso-1.svg)

</Col></Row>

<Row><Col items={2}>

### Orden.
No modificar el **orden** de los estilos.

</Col><Col items={2}>

![completar](/guía-de-estilos/tipografía/mal-uso-2.svg)

</Col></Row>

<Row><Col items={2}>

### Subrayados.
No usar **subrayados** en encabezados.

</Col><Col items={2}>

![completar](/guía-de-estilos/tipografía/mal-uso-3.svg)

</Col></Row>

<Row><Col items={2}>

### Paleta cromática.
No cambiar el **color** de la tipografía.

</Col><Col items={2}>

![completar](/guía-de-estilos/tipografía/mal-uso-4.svg)

</Col></Row>

<Row><Col items={2}>

### Alineación y sangrías..
No alinear el contenido **al centro o a la derecha**. No utilice **sangrías**.

</Col><Col items={2}>

![completar](/guía-de-estilos/tipografía/mal-uso-5.svg)

</Col></Row>

## Extensión tipográfica
La extensión es muy importante a la hora de interpretar un texto. Nuestras referencias recomiendan entre 40 caracteres como mínimo y 84 como máximo, lo que serian entre 4 y 8 columnas. Esta prohibido el uso de texto en extensión mas allá de 8 columnas u 84 caracteres.
