## Unidades
Para utilizar los espaciados se deben seguir las pautas establecidas entre las diferentes clases de componentes. No se deben utilizar otros espaciados ya que pueden perjudicar la lectura del usuario y desorganizar la información.

![completar](/guía-de-estilos/espaciados/espaciados.svg)

## Reglas de espaciado
Mantener esta consistencia en los productos es crucial para el usuario. Esta es la orientación requerida y debe seguirse en todo momento.
