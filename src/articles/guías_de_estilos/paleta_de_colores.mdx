## Paleta primaria

Nuestra paleta principal se compone de neutros, grises y azules, que conforman el grisulado, para aportar sobriedad a las interfaces de Oblesico y utilizamos el amarillo de una forma lógica en todo el producto para guiar el ojo y resaltar los llamados a la acción.

## Colores semánticos

Cada tipo de texto tiene un color asignado. El uso constante del color mantiene baja la carga cognitiva y crea una experiencia de usuario unificada y atractiva.

## Fondos

Los fondos establecen limitaciones dentro de una misma pantalla, pueden utilizarse diferentes fondos para organizar la información de una manera ordenada y prolija.

<Row>
  <Col items={4}>
    <Color name="Blanco" description="Fondos primarios" hex="#ffffff" />
  </Col>
  <Col items={4}>
    <Color name="Grisulado 1" description="Fondos secundarios" hex="#f3f6f9" />
  </Col>
</Row>
