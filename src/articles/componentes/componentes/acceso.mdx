## Tipos de accesos

### Acceso grande

Se utilizan para acceder a secciones de mayor relevancia. Principalmente se usan en bloque (en grupo) para darle al usuario el contexto necesario del/los sitio/s a los que se ingresará.

Los accesos de menor tamaño se usan principalmente para dos instancias: para el patrón de selección de accesos y para cerrar sesión.

## Usabilidad

### Buenas prácticas

[[si | **Incluir pocos elementos en el patrón de selección de accesos.** Esto es para que sea más fácil de escanear con la vista. Limitarse a 7 opciones como máximo.]]

[[si | **Resumir todo en una línea de texto.** Tanto el título del desplegable como el título individual de cada acceso no deben superar los 25 caracteres. Tampoco se deben usar puntos suspensivos para truncarlo.]]

### Malas prácticas

[[no | **No agrupar información con funcionalidades dentro de un mismo patrón de selección de accesos.** Si bien los accesos pueden usarse para ingresar a un sitio con más información (por ejemplo, ver los requisitos de un trámite), como también para realizar una acción (por ejemplo, cerrar sesión), estos dos tipos de accesos no se deben agrupar en un solo patrón. En todo caso, generar otro.]]
