## Tipos de barras de progreso

### Con pasos

Este componente se utiliza cuando se puede determinar el porcentaje de la tarea completada.

<div class="content-2">
  <div class="progression-container">
    <p class="steps">1 de 4 pasos</p>
    <div class="progress">
      <div
        class="progress-bar bg-info"
        role="progressbar"
        aria-valuenow="0"
        aria-valuemin="0"
        aria-valuemax="80"
        style="width: 25%;"
      ></div>
    </div>
  </div>
</div>

### Sin pasos

Este tipo de barra muestra visualmente el progreso de una operación del sistema. Se utiliza cuando se desconocen los pasos para completar esta operación.

<div class="content-2">
  <div class="progression-container">
    <div class="progress">
      <div
        class="progress-bar bg-info"
        role="progressbar"
        aria-valuenow="0"
        aria-valuemin="0"
        aria-valuemax="80"
        style="width: 25%;"
      ></div>
    </div>
  </div>
</div>

## Diseño y estructura (contexto de uso)

Una barra de progreso muestra el estado de finalización de una tarea en curso.

![completar](/componentes/barradeprogreso/barra-progreso.svg)

## Usabilidad

### Buenas prácticas

[[si | **Utilizar una barra de progreso determinada** cuando se pueda calcular el progreso con respecto a un objetivo específico.]]
[[si | **Utilizar una barra de progreso indeterminada** cuando no se pueda determinar el tiempo o el esfuerzo para completarlo.]]
