## Anatomía

![completar](/componentes/paso-a-paso/anatomía.svg)

1. El área de título es la presentación del estado actual.
2. El área descriptiva siguiente, indica el título del paso siguiente.
3. El área representativa conformada por circulos, según la cantidad de pasos, permite mostrar de forma gráfica en que paso se encuentra y cuantos faltan.
4. Los pasos faltantes, es la manera textual de mostrar la cantidad de pasos faltantes y en el que se encuentra.

## Opciones de paso a paso

### determinados

Los paso a paso son indicadores determinados que muestran cuántos pasos llevará un proceso. Deben utilizarse cuando se pueda detectar la tasa de finalización del proceso.

## Usabilidad

**Progreso**: El paso a paso avanza automáticamente a medida que el sistema se carga de forma determinada por el usuario. Las acciones de un usuario afectan el paso a paso, indica cuántos pasos debe progresar para que finalice el proceso.

**Integración con acciones**: El indicador de paso se integra al accionable de progreso, componente botón, para expresar una conexión entre una interacción y un proceso determinado. Por lo general, se usan para expresar una interacción de avance, como el hacer clic en el botón "Siguiente".
