## Anatomía

<!-- ![completar](/componentes/card/anatomía.svg) -->

1. El área de título se utiliza para introducir la información que se encuentra al accionar el colapsable. 
2. El ícono cambia según si el colapsable esta abierto o cerrado. 
3. El área descriptiva es el contenido que se encuentra oculto dentro del colapsable. Una vez que el usuario acciona sobre el área de título, se abrirá el componente mostrando el área descriptiva.

## Opciones de colapsables

### Selección única

Los grupos de colapsables de selección única permiten un solo colapsable abierto por vez. Cuando se acciona otro colapsable dentro del mismo grupo se despliega mostrando el contenido, mientras que el último que estaba abierto procede a cerrarse.

### Tarjetas sin contenedor

Las tarjetas sin contenedor se diferencian del resto por tener más espacio para su “area de títulos” y “área descriptiva”. Además esta tarjeta no es accionable, sino que al final tendrá un enlace que marcará la acción que se realizará.

## Usabilidad

**Todo el área de títulos debe ser seleccionable**. Los usuarios podrán hacer clic en cualquier parte del área para expandir o contraer el contenido. 

**Los colapsables deben respetar la guía de espaciados**. Hay que asegurarse que cada colapsable este suficientemente lejos del siguiente para que los usuarios no se activen accidentalmente. La distancia exacta va a depender del dispositivo.
