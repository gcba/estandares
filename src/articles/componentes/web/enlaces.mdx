## Tipos de enlaces

### Default

Es su estado principal y lo utilizamos para conectar o enlazar un contenido que no se encuentra en la página actual.

<p class="content-2 text-center">
  <a href="#" target="_blank" rel="noreferrer">
    Enlace
  </a>
</p>

### Ancla

Se usan para elementos anclados en una parte de la página que funcionan a modo de índice.


<p class="content-2 text-center d-grid">
  <a href="#">1º ancla-enlace</a>
  <a href="#">2º ancla-enlace</a>
  <a href="#">3º ancla-enlace</a>
  <a href="#">4º ancla-enlace</a>
</p>

### Externo

Identificamos al enlace con un icono para referirnos que el sitio siguiente es un sitio externo al sitio actual.

<p class="content-2 text-center">
  <a href="#" className="external">
    Externo
  </a>
</p>

### Descarga

Identificamos al enlace con un icono de descarga para referirnos que al accionar sobre el podremos descargar un archivo.

<p class="content-2 text-center">
  <a href="#" className="download">
    Descarga
  </a>
</p>

## Diseño y estructura (Contexto de uso)

### Enlace default en párrafos

Pueden variar según el hipervínculo del enlace al que vaya.

![completar](/componentes/enlace/enlace-parrafo.svg)

### Enlace externo en párrafos

Pueden variar según el hipervínculo del enlace al que vaya.

![completar](/componentes/enlace/enlace-externo.svg)

### Enlace anclas

Se ubican al comienzo de un sitio, para anclar a un encabezado de esa misma página.

![completar](/componentes/enlace/enlace-ancla.svg)

### Descarga

Se suelen utilizar al finalizar una pantalla para poder descargar esos archivos relacionados.

![completar](/componentes/enlace/enlace-descarga.svg)
