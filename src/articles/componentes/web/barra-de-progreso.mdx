## Tipos de barra de progreso

### Con pasos

Utilice este componente cuando pueda determinar los pasos de la tarea completada.

<div class="content-2">
  <div class="progression-container">
    <p class="steps">1 de 4 pasos</p>
    <div class="progress">
      <div
        class="progress-bar bg-info"
        role="progressbar"
        aria-valuenow="0"
        aria-valuemin="0"
        aria-valuemax="80"
        style="width: 25%;"
      ></div>
    </div>
  </div>
</div>

### Sin pasos

Este tipo de barra muestra visualmente el progreso de una operación del sistema. Se utiliza cuando se desconoce los pasos para completar la misma.

<div class="content-2">
  <div class="progression-container">
    <div class="progress">
      <div
        class="progress-bar bg-info"
        role="progressbar"
        aria-valuenow="0"
        aria-valuemin="0"
        aria-valuemax="80"
        style="width: 25%;"
      ></div>
    </div>
  </div>
</div>

## Diseño y estructura (contexto de uso)

Se ubican en la parte superior de la página y muestran el estado de una tarea.

![completar](/componentes/barradeprogreso/barra-progreso.svg)


## Usabilidad

### Buenas prácticas

[[si | **Utilice una barra de progreso determinada** cuando el progreso se pueda calcular con respecto a un objetivo específico.]]
[[si | **Utilice una barra de progreso indeterminada** cuando no se pueda determinar el tiempo o el esfuerzo para completarlo.]]
