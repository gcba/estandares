## Usabilidad

### Buenas prácticas

![completar](/bxs-check-circle.svg) **Los enlaces de navegación deben ser cortos**. Pueden ser derivados más cortos de los propios títulos de página. 

![completar](/bxs-check-circle.svg) **Reducir a una o dos palabras** para que nunca sean tan largas que requieran truncamiento. 

![completar](/bxs-check-circle.svg) **En la navegación vertical solo los elementos del primer nivel pueden tener iconos**. 
Cualquier elemento de subnivel siempre debe ser solo de texto para garantizar una jerarquía clara. 

![completar](/bxs-check-circle.svg) **En la navegación vertical si los elementos de navegación de nivel superior tienen una ubicación asociada**, envíe al usuario a esa ubicación y abra los elementos de navegación de subnivel. 

![completar](/bxs-check-circle.svg) **En la navegación vertical si los elementos de navegación de nivel superior no tienen una ubicación asociada**, solo abra los elementos de navegación de nivel inferior. 

![completar](/bxs-check-circle.svg) **La navegación vertical puede usar cualquiera de estos comportamientos, pero nunca debe mezclar comportamientos** en la misma experiencia. 

![completar](/bxs-check-circle.svg) **En la navegación horizontal si hay demasiados componentes** puede permitir el desplazamiento horizontal en responsive. 

![completar](/bxs-check-circle.svg) La navegación horizontal pueden usarse en conjunto con [Botón](https://gcba.github.io/Obelisco/?path=/story/componentes-bot%C3%B3n--colores) 
y [Desplegables](https://gcba.github.io/Obelisco/?path=/story/componentes-desplegables--desplegable-con-icono).
