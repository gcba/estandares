## Anatomía

![completar](/componentes/paginado/anatomía.svg) 

1. Anterior y siguiente: Navega hacia atrás y hacia adelante desde la página seleccionada actualmente.
2. Navegación de página: Navegue a una página específica seleccionando el número de página.
3. Truncamiento: Cuando se alcanza un umbral de páginas, la lista se trunca con puntos suspensivos. Este aparece en el ante último lugar, para mostrar únicamente que existen mas en el medio.
4. Posición actual: Indica la página o vista actual que se este mostrando.

## Opciones de paginado

### Tarjetas verticales u horizontales

Los paginados siempre van centrados con respecto a la página. La cantidad máxima de botones a mostrar es de 9 páginas (incluidos los puntos suspensivos).

## Usabilidad

**Uso principal**: la paginación se usa comúnmente para resultados de búsqueda.

**Posicionamiento**: se encuentra en la parte inferior de la página y permite al usuario moverse fácilmente entre cada página.

**Vision general**: generalmente, se usa si hay más de 25 elementos mostrados en una vista.

**Truncamiento**: cuando el número de páginas supera el límite máximo de visualización, se utiliza una elipsis, puntos suspensivos (…), para truncar las páginas restantes.

**Botón**: cada página del control de paginación es un enlace de botón que les permite navegar rápidamente por los resultados.
