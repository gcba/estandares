## Anatomía

![completar](/componentes/alerta/anatomía-1.svg)

1. El ícono junto con el color van a representar el tipo de alerta. 
2. El área destacable funciona como primer introducción y llamado de atención a lo que se quiere comunicar. 
3. El área descriptiva tiene información más detallada para el usuario.

## Opciones de alertas

### Alertas de error

Las alertas de error se usan para comunicar problemas que deben resolverse, validación de acciones o errores del sistema. Los usuarios leerán un mensaje que les ayudará a resolver y a continuar con su navegación.

<div className="alert-wrapper">
  <div className="alert alert-danger" role="alert">
    <p>
      <strong>Este es un destacado de una alerta de error (opcional). </strong>
      Esta es la descripción de una alerta de error que continua el texto destacado y es obligatoria.
    </p>
  </div>
</div>

<div className="alert-wrapper">
  <div className="alert alert-danger alert-dismissible fade show" role="alert">
    <p>
      <strong>Este es un destacado de una alerta de error. </strong>
    </p>
    <ol>
      <li>
        <a href="#">Ancla al error 1</a>
      </li>
      <li>
        <a href="#">Ancla al error 2</a>
      </li>
      <li>
        <a href="#">Ancla al error 3</a>
      </li>
    </ol>
  </div>
</div>


### Alertas de éxito

Las alertas de éxito funcionan para validación de acciones del usuario. Por ejemplo como confirmación de una tarea se completó correctamente.

<div className="alert alert-success" role="alert">
    <p>
      <strong>Este es un destacado de una alerta de éxito (opcional). </strong>
      Esta es la descripción de una alerta de éxito y es obligatoria.
    </p>
</div>


### Alertas de información

Las alertas de información se usan para notificar a los usuario sobre un cambio, para darles consejos o brindar información especifica sobre la sección en la que se encuentran. Son las únicas alertas que la persona tiene habilitada la acción de cerrarlas en caso de que considere que es contenido molesto o innecesario.

<div className="alert-wrapper">
  <div className="alert alert-info alert-dismissible fade show" role="alert">
    <p>
      <strong>Este es un destacado de una alerta de información (opcional). </strong>
      Esta es la descripción de una alerta de información y es obligatoria.
    </p>
    <button type="button" data-dismiss="alert" className="btn btn-sm btn-info close-btn">
      Cerrar
    </button>
  </div>
</div>


### Alertas de advertencia

Las alertas de advertencia se usan para mostrar información que necesita la atención de los usuarios. Este tipo de alertas influyen en la navegación del usuario, por lo que no tienen cierre y es muy importante que las tengan en cuenta.

<div className="alert alert-primary" role="alert">
  <p>
    <strong>Este es un destacado de una alerta de advertencia (opcional). </strong>
    Esta es la descripción de una alerta de advertencia y es obligatoria.
  </p>
</div>

## Usabilidad
**Comprender el contexto del usuario**. No incluir notificaciones que no estén relacionadas con el objetivo actual del usuario.

**Que el usuario descarte una alerta cuando sea apropiado**.

**Las alertas son una oportunidad**. Los usuarios leerán un mensaje que les ayudará a resolver un error incluso si por lo general no leen la documentación; incluya contenido que ayude al usuario a resolver.
