## Tipos de alertas

Las opciones de alerta muestran su color semántico e ícono correspondiente para ayudar a las personas con deficiencia de visión del color a discernir el tono del mensaje.

### Alertas de error

Se usan para mostrar un error de validación, falla del sistema, o para transmitir algo que debe reconocerse o abordarse de inmediato para continuar con su navegación.

<div className="alert-wrapper">
  <div className="alert alert-danger" role="alert">
    <p>
      <strong>Este es un destacado de una alerta de error. </strong>
      Esta es la descripción de una alerta de error que continua el texto destacado y es obligatoria.
    </p>
  </div>
</div>

<div className="alert-wrapper">
  <div className="alert alert-danger alert-dismissible fade show" role="alert">
    <p>
      <strong>Este es un destacado de una alerta de error. </strong>
    </p>
    <ol>
      <li>
        <a href="#">Ancla al error 1</a>
      </li>
      <li>
        <a href="#">Ancla al error 2</a>
      </li>
      <li>
        <a href="#">Ancla al error 3</a>
      </li>
    </ol>
  </div>
</div>

### Alertas de éxito

Se usan para informar al usuario sobre una tarea que se realizó con éxito.

<div className="alert alert-success" role="alert">
  <p>
    <strong>Este es un destacado de una alerta de éxito. </strong>
    Esta es la descripción de una alerta de éxito y es obligatoria.
  </p>
</div>

### Alertas de información

Notifican al usuario sobre un cambio, para darles consejos o brindar información específica sobre la sección en la que se encuentran.

<div className="alert-wrapper">
  <div className="alert alert-info alert-dismissible fade show" role="alert">
    <p>
      <strong>Este es un destacado de una alerta de información. </strong>
      Esta es la descripción de una alerta de información y es obligatoria.
    </p>
  </div>
</div>

### Alertas de advertencia

Esta variante se utiliza para informar sobre una situación que debe abordarse pronto.

<div className="alert alert-primary" role="alert">
  <p>
    <strong>Este es un destacado de una alerta de advertencia. </strong>
    Esta es la descripción de una alerta de advertencia y es obligatoria.
  </p>
</div>

## Diseño y estructura (Contexto de uso)

### En cuerpo de página

Los carteles de alerta están destinados a llamar la atención y pedir a los usuarios que tomen medidas necesarias en el contexto de una página. Se colocan al principio de aquello que se quiera advertir, alineados al margen izquierdo.
