## Anatomía

![completar](/componentes/breadcrumb/anatomía-1.svg)

1. El enlace describe la relación jerárquica entre la página actual y la estructura general del sitio. Ayuda a posicionar al usuario y permite interactuar para la navegación del sitio.
2. El ícono se utiliza para indicar el recorrido del sitio para llegar la página actual.


## Opciones de miga de pan

Las rutas de navegación muestran dónde se encuentra la página actual en la jerarquía del sitio web. Utilizar migas de pan refuerza la estructura del sitio.

## Usabilidad

**Empiece con la palabra "Inicio" como primer enlace de las rutas de navegación**.

**Considerar el tamaño de los botones**. Aunque las rutas de navegación se muestran con frecuencia con texto más pequeño, asegúrese de que el texto sea el tamaño dispuesto.
