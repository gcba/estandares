## Tipos de etiquetas

### Default

Cuando utilizamos las etiquetas en un mismo componente no las diferenciamos por color, sino por texto. En estos casos usamos etiquetas default.

<div className="d-flex flex-row justify-content-center content-1">
  <span className="badge badge-secondary">Etiqueta</span>
</div>

Cuando las etiquetas tienen un significado semántico se diferencian por colores. Utilizar estas variantes para los siguientes estados:

### Éxito

De éxito para casos aprobados, completos, exitosos, finalizados.

<div className="d-flex flex-row justify-content-center content-1">
  <span className="badge badge-success">Etiqueta</span>
</div>

### Información 

De información para casos en uso, en vivo, publicados, nuevos.

<div className="d-flex flex-row justify-content-center content-1">
  <span className="badge badge-info">Etiqueta</span>
</div>

### Advertencia

De advertencia para casos de alerta, fallidos, en pausa, rechazado

<div className="d-flex flex-row justify-content-center content-1">
  <span className="badge badge-primary">Etiqueta</span>
</div>

### Peligro

De peligro para casos de error, no completados, eliminados.

<div className="d-flex flex-row justify-content-center content-1">
  <span className="badge badge-danger">Etiqueta</span>
</div>

## Diseño y estructura (contexto de uso)

Dentro de las tarjetas se pueden usar un máximo de tres etiquetas.

![completar](/componentes/etiqueta/en_tarjetas.png)

### En texto

**Siempre deben estar alineadas a la izquierda,** con el objetivo de que se mantenga un orden de lectura natural.

![completar](/componentes/etiqueta/en_texto.svg)

**Aparecen al principio de un artículo con una sola etiqueta principal y/o al final del texto** como parte de una sección de “Etiquetas relacionadas”.

![completar](/componentes/etiqueta/en_texto2.svg)

## Usabilidad

### Buenas prácticas

[[si | **Deshabilitar los diferentes estados cuando la etiqueta es estática.** Si la etiqueta es meramente informativa y no se encuentra asociada a un enlace de destino es necesario deshabilitar el hover, on-click y focus.]]

[[si | **Usar las etiquetas con moderación.** Al combinar las etiquetas con otros elementos visuales se puede generar mucho ruido visual en la pantalla, por lo que es preferible usarlas únicamente si le es útil a la persona ciudadana.]]

### Malas prácticas

[[no | **No combinar etiquetas estáticas con interactivas.** Evitar que una etiqueta con enlace se ubique al lado de otra que no tenga ese atributo. Optar por una de las dos, según sea más pertinente.]]
