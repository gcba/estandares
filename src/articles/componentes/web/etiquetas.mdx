## Tipos de navegación

### Default

Cuando las etiquetas las utilizamos en un mismo componente no las diferenciamos por color, sino por texto. En estos casos usamos etiquetas default.

  <span className="badge badge-secondary">Etiqueta</span>
</div>

### Éxito

De éxito para casos aprobados, completos, exitosos, finalizados.

</div>

## Diseño y estructura (Contexto de uso)

### En tarjetas

Dentro de las tarjetas se pueden usar un máximo de tres etiquetas.

![completar](/componentes/etiqueta/en_tarjetas.png)

### En texto

**Siempre deben estar alineadas a la izquierda,** con el objetivo de que se mantenga un orden de lectura natural.

![completar](/componentes/etiqueta/en_texto.svg)

**Aparecen al principio de un artículo, usando una sola etiqueta principal y/o al final,** como parte de una sección con encabezado “Etiquetas relacionadas”, luego del final del texto.

![completar](/componentes/etiqueta/en_texto2.svg)

## Usabilidad

### Buenas prácticas

[[si | **Las etiquetas aumentan la cantidad de ruido visual** así que utilícelas con moderación.]]

[[si | **Si requiere varias etiquetas, considere reservar una sola** para mostrar el estado, la categoría o el significado más importante.]]

[[si | **Los usuarios suelen confundir etiquetas con botones.** Evite las etiquetas si aparecen en la misma área de la página que los botones.]]

[[si | **Reducir a una o dos palabras** para que nunca sean tan largas que requieran truncamiento.]]
