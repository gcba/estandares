## Tipos de migas de pan

Las migas de pan muestran dónde se encuentra la página actual en la jerarquía del sitio web. Utilizar migas de pan refuerza la estructura del sitio.

### Desktop

Las rutas de navegación muestran dónde se encuentra la página actual en la jerarquía del sitio web. Utilizar migas de pan refuerza la estructura del sitio. En desktop, se ve completo.

<div className='d-flex flex-row justify-content-center content-1'>
  <nav aria-label="breadcrumb">
    <ol className="breadcrumb">
      <li className="breadcrumb-item">
        <a href="#">Inicio</a>
      </li>
      <li className="breadcrumb-item">

      </li>

      </li>

  </nav>

### Mobile

Mientras que en dispositivos móviles, se ve de manera reducida, solo para permitir un regreso.

<div className='d-flex flex-row justify-content-center content-1'>
  <nav aria-label="breadcrumb">
    <ol className="breadcrumb">
      <li className="breadcrumb-item"><a href="#">Buenos Aires</a></li>
      <li className="breadcrumb-item"><a href="#">Página anterior</a></li>
    </ol>
  </nav>
</div>

## Diseño y estructura (contexto de uso)

Se ubican sobre el título de la página y muestran el progreso jerárquico desde el nivel de elemento más alto hasta el más bajo. El último de ellos muestra la página actual y no es enlace.

Los niveles que posean más de 30 caracteres aproximadamente se truncarán y agregarán tres puntos (…) al final para visualizar que hay más texto en dicho nivel.
