## Tipos de accesos

### Acceso grande

Se utilizan para acceder a secciones de mayor relevancia. Principalmente se usan en bloque (en grupo) para darle al usuario el contexto necesario del/los sitio/s a los que se ingresará.

  </div>
</div>

### Acceso chico

Los accesos de menor tamaño se usan principalmente para dos instancias: para el patrón de selección de accesos y para cerrar sesión.

## Diseño y estructura (Contexto de uso)

Esta variante se usa en el caso de que necesitemos agrupar una serie de accesos pequeños que no van a depender de otro componente.

### Malas prácticas


<div class="d-flex flex-row justify-content-start">
  <i class='bx bx-sm bxs-x-circle' style='color:#d02323'></i>
  <p class="pl-2">
    <strong>Dentro de un mismo patrón de selección de accesos </strong> 
    no agrupar información con funcionalidades. Si bien los accesos pueden ser usados para ingresar a un sitio con información (como ver los requisitos de un trámite), como también para realizar una acción (como cerrar sesión), estos dos tipos de accesos no se deben agrupar en un solo patrón. En todo caso, generar otro.
  </p>
</div>
