## Anatomía

![completar](/componentes/modals/anatomia-1.svg)

1. El Área de título se componte de un contenido titular que engloba el contenido.
2. El área descriptiva muestra el contenido en relación al título que lo abarca, puede ser una pregunta que se relacione con el Área accionable.
3. El Área accionable se compone de dos (2) botones, con una acción positivia-activa y otra negativa o de cancelación.

## Usabilidad

**Requerir una respuesta inmediata del usuario**. Utilice un modal para solicitar información que impida que el sistema continúe con un proceso iniciado por el usuario.

**Notificar al usuario de información urgente**. Utilice un modal para notificar al usuario de información urgente sobre su trabajo actual. Los modales se utilizan comúnmente para confirmar acciones o transmitir una consecuencia de la acción de un usuario.

**Confirmar la decisión de un usuario**. Utilice un modal para confirmar las decisiones del usuario. Describa claramente la acción que se está confirmando y explique las posibles consecuencias que pueda causar. Tanto el título como el botón deben reflejar la acción que ocurrirá, si la acción es destructiva o irreversible.
