## Tipos de controles de formularios

### Campo de texto simple

Los campos de texto simples pueden llevar un placeholder para mostrar un ejemplo de los datos que se están solicitando. El área de ayuda proporciona información extra de asistencia.

<div class="content-2">
  <div class="form-wrapper">
    <div class="form-group">
      <label for="name-input">Nombre</label>
      <input type="text" class="form-control" id="name-input" name="name" />
    </div>
  </div>
  <div class="form-wrapper bg-light p-4">
    <div class="form-group">
      <label for="name-input">Nombre</label>
      <input type="text" class="form-control" id="name-input" name="name" />
    </div>
  </div>
</div>

### Selector

Los selectores se utilizan cuando el sistema ya posee los datos que se precisan y el usuario solo debe seleccionar uno de ellos. Debe haber al menos cinco opciones de selección.

<div class="content-2">
  <div class="form-wrapper">
    <div class="form-group">
      <label for="lista-de-seleccion">Nombre del selector *</label>
      <select class="custom-select" id="lista-de-seleccion">
        <option>1</option>
        <option>2</option>
        <option>3</option>
        <option>4</option>
        <option>5</option>
      </select>
    </div>
  </div>
  <div class="form-wrapper bg-light p-4">
    <div class="form-group">
      <label for="lista-de-seleccion">Nombre del selector</label>
      <select class="custom-select" id="lista-de-seleccion">
        <option>1</option>
        <option>2</option>
      </select>
    </div>
  </div>
</div>

### Campo de búsqueda

Los campos de búsqueda llevan siempre un placeholder que anticipe y ayude al usuario a saber que información podrá encontrar en el buscador.

  <div class="form-wrapper mb-3">
    <form class="form-search">
      <div class="form-group">
        <input type="search" class="form-control input-search" id="name-input" name="name" placeholder="Buscar..." />
        <button class="reset" type="reset"></button>
      </div>
    </form>
  </div>
  <div class="form-wrapper bg-light p-4">
    <form class="form-search">
      <div class="form-group">
        <input type="search" class="form-control input-search" id="name-input" name="name" placeholder="Buscar..." />
        <button class="reset" type="reset"></button>
      </div>
    </form>
  </div>
</div>


### Campo de fecha

Los campos de fecha siempre llevan un placeholder que indiquen al usuario como debe completar el campo. Cuando se acciona el campo, la fecha se completa mediante el teclado.


<div class="content-2">
  <div class="form-wrapper">
    <div class="form-group">
      <label for="date-input">Fecha</label>
      <span id="date-help" class="form-text">
        Nombre del campo *
      </span>
      <input type="date" class="form-control" id="date-input" name="date" aria-describedby="date-help" />
    </div>
  </div>
  <div class="form-wrapper bg-light p-4">
    <div class="form-group">
      <label for="date-input">Fecha</label>
      <span id="date-help" class="form-text">
        Nombre del campo *
      </span>
      <input type="date" class="form-control" id="date-input" name="date" aria-describedby="date-help" />
    </div>
  </div>
</div>


### Área de texto

El área de texto puede tener o no una limitación de caracteres. Cuando hay una limitación se visualiza con la cantidad de caracteres que hay actualmente y la cantidad máxima permitida.


<div class="content-2">
  <div class="form-wrapper">
    <div class="form-group">
      <label for="text-area">Bloque de texto</label>
      <textarea class="form-control" id="text-area" rows="3"></textarea>
    </div>
  </div>
  <div class="form-wrapper bg-light p-4">
    <div class="form-group">
      <label for="text-area">Bloque de texto</label>
      <textarea class="form-control" id="text-area" rows="3"></textarea>
    </div>
  </div>
</div>

### Campo numérico

Los campos numéricos pueden cargarse tanto por teclado como por las flechas indicadoras. Este tipo de campos se utilizan cuando necesitamos un valor de cantidad.


<div class="content-2">

      <label for="number-input">Campo numérico *</label>
      <input type="number" class="form-control" id="number-input" name="number" value="30" />
    </div>
  </div>
  <div class="form-wrapper bg-light p-4">
    <div class="form-group">
      <label for="number-input">Campo numérico *</label>
      <input type="number" class="form-control" id="number-input" name="number" value="30" />

### Radio button

Permite seleccionar una sola opción de una lista de opciones. Lo recomendable es que el usuario pueda ver todas las opciones al mismo tiempo y compararlas.


import { RadioInput } from "../../../components/docs/forms/RadioInput"

<div class="content-2">
  <div class="form-wrapper d-grid text-center">
    <RadioInput
      name="skills"
      options={[
        { value: "uno", label: "Opción 1" },
        { value: "dos", label: "Opción 2" },
        { value: "tres", label: "Opción 3" },
      ]}
    />
  </div>
</div>

## Diseño y estructura (Contexto de uso)

Un formulario permite a los usuarios ingresar información en una página. Los botones deben estar ordenados por importancia de derecha a izquierda, con el botón más importante (principal) a la derecha. Agrupe las secciones relacionadas en conjuntos lógicos con un encabezado o divídalo en otra página, para que los usuarios puedan comprender la información que necesitan ingresar.
