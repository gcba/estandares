## Opciones de tablas

Las tablas ayudan a organizar la información de manera lógica y agrupar cosas similares. Ayudan a los usuarios a encontrar información específica dentro de un gran conjunto de datos.

## Usabilidad

**Alinear a la izquierda datos textuales.** Los datos textuales siempre están alineados a la izquierda dentro de una tabla. La alineación central nunca debe usarse.

**Alinear a la derecha datos numéricos.** Los datos numéricos siempre están alineados a la derecha dentro de una tabla. La alineación central nunca debe usarse. Solo debe alinearse a la izquierda cuando los números son identificadores arbitrarios, conocidos como "números nominales", lo que significa que no se pueden comparar o combinar aritméticamente (por ejemplo, códigos postales, direcciones IP, números de teléfono). Los encabezados de columna siguen la alineación de los datos.

**Respete la alineación vertical.** Todo el contenido debe estar centrado verticalmente dentro de la fila para ofrecer el equilibrio visual adecuado.

**Utilice un guion (-) para los valores faltantes.** Cuando haya brechas en los datos, use un guión corto (-) para representar valores nulos o no aplicables (N / A).
