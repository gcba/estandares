## Opciones de tablas


Las tablas ayudan a organizar la información de manera lógica y agrupar cosas similares. Ayudan a los usuarios a encontrar información específica dentro de un gran conjunto de datos.

## Usabilidad

**Empiece con la palabra "Inicio" como primer enlace de las rutas de navegación.** 

**Considerar el tamaño de los botones.** Aunque las rutas de navegación se muestran con frecuencia con texto más pequeño, asegúrese de que el texto sea el tamaño dispuesto.
