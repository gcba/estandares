## Anatomía

![completar](/componentes/card/anatomía.svg)

1. El área preliminar puede estar conformada por imágenes o iconos.
2. El área de títulos esta conformado por títulos y subtítulos.
3. El área descriptiva es una bajada que introduce contenido que se encontrará luego.
4. El área de cierre es una información extra que puede ser de interés.

## Opciones de tarjetas

## Diseño y estructura

### Ancho de tarjetas

Al pensar como ubicar las tarjetas en la composición debemos considerar que queremos comunicar, usando textos simples y directos. En el caso de utilizar cuatro tarjetas en una misma fila, estas mismas no pueden llevar descripción, solo podrán tener un “Área preliminar” y un “Área de títulos” ya que el espacio no es optimo para insertar una descripción. Por eso se debe elegir muy bien que título van a llevar estas tarjetas.

![completar](/componentes/card/estructura-1.svg)

Para tarjetas con un “Área descriptiva” van a ser tres la cantidad máxima a utilizar por fila.

## Usabilidad

**No usar tarjetas solo para decoración**. El componente de tarjetas es para tarjetas, no para cualquier tipo de contenido que necesite un borde alrededor.

**Las tarjetas son accionables**. Se utilizan como un resumen de información más detallada, cualquier tarjeta individual debe vincularse a esa información.

**No incluir contenido redundante**. Las tarjetas tienen que comunicar lo que vas a encontrar una vez que el usuario acciona.

**Simpleza**. Se tiene que evitar todo tipo de elemento que distraiga al usuario de la acción que quiere realizar.

**Versatilidad**. Se dispone de varios elementos para componer una tarjeta, por eso usarlos en el orden establecido es necesario.
