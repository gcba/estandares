## Anatomía

![completar](/componentes/listas/anatomia-1.svg)

1. Las secciones mas importantes ocupan el nivel 1, estas pueden no estar ordenadas alfabéticamente.
2. Las secciones secundarias se encuentran en nivel 2, y estas también pueden no estar ordenadas alfabéticamente.
3. Las secciones de nivel 3 recomendamos que estén ordenadas alfabéticamente.

## Opciones de listas

### Niveles

Los componentes de listas están diseñados para agregar niveles según sea necesario. Las listas como maximo se componen de hasta 3 niveles. Si se tienen menos páginas se recomienda reorganizar la información buscando otras alternativas.

## Usabilidad

**Muestra la página actual**. Indique dónde se encuentra un usuario dentro de la navegación. Utilice el estado "seleccionado" para mostrar a los usuarios a qué página han navegado.

**Mantenga breves los enlaces de navegación**. Pueden ser derivados más cortos de los propios títulos de página.

**Siempre pruebe su navegación**. Si la jerarquía de navegación es demasiado larga, los usuarios pueden perder elementos en la parte inferior. Si es demasiado profundo, los usuarios pueden perder elementos que requieren demasiados clics. Realice una prueba de usabilidad para encontrar el equilibrio adecuado entre amplitud y profundidad.

**Asegúrese de que el sistema de navegación lateral sea accesible desde el teclado**. Los usuarios deberían poder navegar a través de cada enlace con el tabulador.
